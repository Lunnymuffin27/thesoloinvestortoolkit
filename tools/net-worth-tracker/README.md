# Net Worth Tracker (MVP)

A lightweight, privacy-first net worth tracker built for The Solo Investor.

## Features
- Track assets and liabilities
- Category breakdowns
- Net worth over time (snapshots + chart)
- CSV export
- Local-only storage (no accounts, no uploads)

## How it works
All data is stored in your browser using localStorage.
Nothing is sent to a server.
